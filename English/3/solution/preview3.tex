\section{学习建议}

\begin{itemize}
  \item \textbf{重点记搭配，而非单词本身。} 例如：\textit{justify a conclusion} 比单记 \textit{justify} 更贴近考试语境。
  \item \textbf{注意学术语体表达。} 如：
    \begin{itemize}
      \item enhance → 提升
      \item derive from → 源自
      \item inherent → 内在的
    \end{itemize}
  \item \textbf{中译英时主动使用这些词。} 即使能用更简单的表达，也尽量替换为学术词，以提升译文的学术风格。
\end{itemize}
